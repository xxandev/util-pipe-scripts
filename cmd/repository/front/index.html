Pipe-Scripts

Run application (launch must be done with superuser privileges):

./pipe-scripts -host ':8090' -scripts-path '/root/.scripts' -wiki-path '/root/.wiki'

Use application (OS Linux):

- exec simple script

invoking a script through a web application

curl "http://localhost:8090/script-exec?script=print.sh"

equivalent of invoking a script in linux

sudo su -
cd /root/.scripts
./print.sh

- exec script with parameters

invoking a script through a web application

curl "http://localhost:8090/script-exec?script=test.py&params=val-1%2Cval-2%2Cval-3%2C%22text%20text%20text%22"

equivalent of invoking a script in linux

sudo su -
cd /root/.scripts
python3 ./test.py val-1 val-2 val-3 "text text text'
